# Docker

## Docker 팁

### docker 에 ssh 로 접근

실행 중인 virtualbox 게스트 OS 확인
```
docker-machine ls
```

virtualbox 게스트 OS (ex: default) 에 ssh 접근
```
docker-machine ssh default
```

### virtual 게스트 OS 셧다운
```
sudo shutdown -h now
```

### sudo 로 root 로그인
```
sudo -i
```
